**arXiv**: https://arxiv.org/abs/2604.27660

**HF Paper**: https://hf-mirror.com/papers/2604.27660  
**HF Votes**: 117

## 简要摘要

*自然语言处理;上下文学习;技能自演化框架*

背景/任务：许多任务需要语言模型从超出其参数知识的复杂上下文中进行推理，这要求模型具备直接从上下文学习知识的能力。

# Hierarchical Abstract Tree for Cross-Document Retrieval-Augmented Generation

**【跨文档检索增强生成树框架】**

**arXiv**: https://arxiv.org/abs/2605.00529  
**AlphaXiv**: https://www.alphaxiv.org/zh/overview/2605.00529  
**HF Paper**: https://hf-mirror.com/papers/2605.00529  
**HF Votes**: 2

## 简要摘要

*自然语言处理;检索增强生成;跨文档多跳问答*

现有树状检索增强生成方法在应对跨文档多跳问题时面临分布适应性差、结构孤立和抽象粒度粗的挑战。

# PhysicianBench: Evaluating LLM Agents in Real-World EHR Environments

**【评估医疗LLM智能体在真实电子病历环境中的表现】**

**arXiv**: https://arxiv.org/abs/2605.02240  
**AlphaXiv**: https://www.alphaxiv.org/zh/overview/2605.02240  
**HF Paper**: https://hf-mirror.com/papers/2605.02240  
**HF Votes**: 3

## 简要摘要

*自然语言处理;智能体评估;医疗任务执行基准*

现有医疗智能体评测多集中于静态知识回忆或简单动作，无法反映真实临床长流程、复合型工作流的需求。
我们提出了PhysicianBench基准，包含100个源自真实临床咨询的长流程任务，并在模拟真实电子病历环境中通过工具调用和结构化检查点评估了多个LLM智能体的能力，结果表明当前模型与真实临床工作流需求存在巨大差距。
