# Learning while Deploying: Fleet-Scale Reinforcement Learning for Generalist Robot Policies

**【机器人持续部署学习】**

**arXiv**: https://arxiv.org/abs/2605.00416  
**AlphaXiv**: https://www.alphaxiv.org/zh/overview/2605.00416  
**HF Paper**: https://hf-mirror.com/papers/2605.00416  
**HF Votes**: 6

## 简要摘要

*机器人；强化学习；多任务操作策略*

通用机器人策略在真实部署中面临分布偏移、长尾故障等挑战，离线数据不足以支撑鲁棒应用。
提出了一个基于机器人机群的离线到在线强化学习框架，通过共享实际部署经验与人类干预数据，持续改进通用视觉语言动作策略，并在真实机器人任务上验证了其性能提升。

# Let ViT Speak: Generative Language-Image Pre-training

**【训练ViT直接生成语言描述】**

**arXiv**: https://arxiv.org/abs/2605.00809  
**AlphaXiv**: https://www.alphaxiv.org/zh/overview/2605.00809  
**HF Paper**: https://hf-mirror.com/papers/2605.00809  

## 简要摘要

*计算机视觉;多模态学习;视觉语言预训练*

本文旨在改进多模态大语言模型中的视觉编码器与语言模型的协同。
提出GenLIP框架，让视觉变换器通过语言建模目标直接预测文本标记，无需额外文本解码器，在多模态任务上取得优异性能。
