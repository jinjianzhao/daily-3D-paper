# HP-Edit: A Human-Preference Post-Training Framework for Image Editing

**【人类偏好对齐的图像编辑后训练框架】**

**arXiv**: https://arxiv.org/abs/2604.19406  
**AlphaXiv**: https://www.alphaxiv.org/zh/overview/2604.19406  
**HF Paper**: https://huggingface.co/papers/2604.19406  
**HF Votes**: 3

## 简要摘要

*计算机视觉；图像生成与编辑；基于人类反馈的扩散模型对齐*

当前图像编辑模型缺乏基于人类偏好的对齐，限制了其实用性。
提出了HP-Edit框架和RealPref-50K数据集，利用偏好评分器和强化学习后训练来提升编辑模型与人类偏好的一致性。

## 深度解析

这篇论文的核心内容，用通俗易懂的方式为你解读如下：

### (1) 任务是什么
*   **任务名称**：基于文本指令的图像编辑。
*   **任务输入**：一张原始图片 + 一条文本编辑指令（例如：“把图中的狗换成猫”、“把背景换成海滩”、“给汽车换个颜色”）。
*   **任务输出**：一张编辑后的新图片，它应该遵循输入指令，同时看起来自然、美观、符合人类审美。
*   **任务功能**：让用户能够方便、快速、高质量地修改图片内容，满足创意设计、产品展示、内容创作等需求。

### (2) 想解决什么问题
现有的图像编辑模型（如Qwen-Image-Edit， FLUX.1-Kontext）虽然功能强大，但它们通常通过在海量、来源混杂的数据上进行“监督微调”来学习。这导致两个**核心问题**：
1.  **审美不匹配**：模型学到的“好编辑”标准，可能与真实世界中人类的审美偏好不一致（比如，合成的图片风格、色彩可能不自然）。
2.  **数据成本高**：要收集大量能反映人类偏好的高质量编辑数据（比如，让人给不同编辑结果打分），需要耗费巨大的人力物力，难以规模化。

简单说，这篇论文想让AI的图片编辑结果**更符合人类觉得“好看、自然”的标准**，并且用一种**高效、低成本**的方法来实现这个目标。

### (3) 核心思路
论文的核心思路是 **“用强化学习（RL）来微调模型，让它学会人类的偏好”** ，并巧妙地解决了“如何低成本获得人类偏好反馈”的难题。具体分为三步：

1.  **造一个“AI评委”（HP-Scorer）**：
    *   先让真人给少量（比如几千张）编辑图片的质量打分（1-5分）。
    *   然后，用一个已经训练好的**视觉大模型（VLM，如Qwen2.5-VL）** 来学习人类的打分标准。通过设计专门的“评分提示词”，让这个VLM学会像人类一样评判图片编辑的好坏。这个VLM就是“HP-Scorer”。

2.  **用“AI评委”高效筛选训练数据**：
    *   用现有的图像编辑模型，在大量未标注的数据上生成编辑结果。
    *   让“AI评委”（HP-Scorer）自动给这些结果打分，并**专门挑出那些“难”的案例**（比如得分不高、容易出错或不符合偏好的案例）。这样就低成本地构建了一个大规模的、富含“偏好信号”的数据集（**RealPref-50K**）。

3.  **用“AI评委”指导模型进化（RL训练）**：
    *   把“AI评委”（HP-Scorer）当作**奖励函数**。
    *   对基础的图像编辑模型进行**强化学习后训练**。训练目标是：当模型生成的编辑结果让“AI评委”打分更高（即更符合人类偏好）时，就获得奖励；反之则受到惩罚。通过这种方式，引导模型调整自己的参数，使其输出逐渐向人类偏好的高质量编辑结果靠拢。

**整个框架叫做HP-Edit**，它的巧妙之处在于**用少量人工标注训练出一个可靠的“AI评委”，然后用这个“AI评委”自动化地完成后续大规模数据筛选和模型训练**，极大地降低了成本。

### (4) 结果怎么样
*   **定量与定性结果**：在多个图像编辑任务（如物体替换、背景替换、调色等）上，经过HP-Edit框架训练后的模型（如Qwen-Image-Edit-2509），其输出在**人类偏好评分**和**视觉质量**上，都显著超过了原始的、未经偏好对齐的基线模型。
*   **用户研究**：真人用户评估也表明，HP-Edit生成的图片更受青睐。
*   **贡献**：
    *   提出了HP-Edit这个有效的偏好对齐后训练框架。
    *   发布了**RealPref-50K**数据集（包含5万多个覆盖8种常见编辑任务的案例）。
    *   发布了**RealPref-Bench**评测基准，方便未来研究。

### (5) 模型结构与流程
*   **基座模型**：论文使用了现有的、功能强大的**扩散模型**作为基础图像编辑模型，例如 **FLUX.1-Kontext** 或 **Qwen-Image-Edit-2509**。这些模型本身已经能很好地理解指令并执行编辑。
*   **整体流程（结合图2）**：
    1.  **输入**：用户提供一张**原始图片**和一条**文本编辑指令**。
    2.  **基座模型推理**：这个指令和图片被送入**基座图像编辑扩散模型**，模型会生成一个初步的编辑结果。
    3.  **“AI评委”评估（HP-Scorer）**：生成的编辑结果，连同原始图片和指令，一起被送入训练好的**视觉大模型（VLM）评委**。评委根据它对人类偏好的理解，给这个编辑结果打一个分数。
    4.  **强化学习更新**：在**训练阶段**，这个分数被用作**奖励信号**。通过**Flow-GRPO**（一种针对扩散模型的强化学习算法）来更新基座模型的参数。模型学习的方向是：**调整自己的生成过程，使得最终输出的图片能从这个“AI评委”那里获得更高的分数**。
    5.  **输出**：经过多轮迭代训练后，模型进化完毕。在**使用阶段**，用户输入图片和指令，模型直接输出**经过人类偏好对齐的、更高质量**的编辑后图片。

**总结一下**：这篇论文的核心是设计了一套“**培养AI审美**”的系统。它先教一个视觉大模型学会人类的打分标准（AI评委），然后让这个评委去指导一个专业的图片编辑AI（扩散模型），告诉它“你这样改不好看，那样改才好看”，从而让编辑AI的作品越来越符合人类的喜好。整个过程利用了AI来自动化处理数据，既有效又节省成本。
