# Prox-E: Fine-Grained 3D Shape Editing via Primitive-Based Abstractions

## 简要摘要

*计算机图形学；三维形状编辑；基于原语的形状编辑*

当前基于文本驱动的三维编辑方法难以在保持物体整体身份的同时进行精细的局部结构修改。
我们提出了训练无关的Prox-E框架，通过几何原语抽象来指导生成模型，实现精细可控的三维编辑。
